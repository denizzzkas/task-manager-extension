# Task Planner — Imperal Extension

Manage tasks with deadlines, priorities and reminders inside Imperal Cloud OS.

## Features

- Create, update, complete and delete tasks
- Priorities: low / medium / high
- Deadlines with overdue detection
- Hourly reminders for tasks due within 24 hours
- Search and export (JSON / CSV)
- Sidebar panel with stats and tabs (All / Active / Completed / Overdue)

## Usage

Talk to the assistant naturally:

- "Create a task: prepare report by Friday, high priority"
- "Show my overdue tasks"
- "Mark task X as done"
- "Export all tasks as CSV"

## Tech

Built with [Imperal SDK](https://github.com/imperal/imperal-sdk) · Python 3.11 · `claude-haiku-4-5`
